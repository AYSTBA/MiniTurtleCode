# Mini Turtle Code

基于Python与Qt6框架开发的原生桌面端Python海龟绘图专用代码编辑器，专为编程入门教育与教学场景设计。

## 功能特点

### 核心功能
- **Python代码编辑**：支持语法高亮、行号显示、自动缩进
- **项目文件管理**：内置文件资源管理器，支持文件操作
- **一键代码运行**：F5快捷键运行代码，实时显示输出
- **海龟绘图**：集成turtle绘图功能，支持实时渲染
- **终端输出**：实时显示程序输出和错误信息
- **网页嵌入**：内置网页视图，支持加载官网和赛事页面

### 界面布局
- **顶部**：标准菜单栏，包含文件、编辑、视图操作
- **左侧**：导航栏，包含文件资源管理器、官网入口、赛事入口
- **中部**：代码编辑区，支持Python语法高亮
- **右侧**：海龟绘图预览区
- **底部**：输出终端，显示运行结果

## 快捷键

- **新建文件**：Ctrl+N
- **打开文件**：Ctrl+O
- **保存文件**：Ctrl+S
- **另存为**：Ctrl+Shift+S
- **撤销**：Ctrl+Z
- **重做**：Ctrl+Y
- **复制**：Ctrl+C
- **粘贴**：Ctrl+V
- **全选**：Ctrl+A
- **运行代码**：F5
- **显示/隐藏终端**：F12
- **清空画布**：Ctrl+Shift+R

